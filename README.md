# SentinelShield - Web Application Firewall (WAF)

## 📌 Description
This project simulates a Web Application Firewall that detects common web attacks like SQL Injection and Cross-Site Scripting (XSS).

## 🎯 Objectives
- Detect SQL Injection attacks
- Detect XSS attacks
- Analyze HTTP requests

## 🛠️ Tools Used
- Python
- Regular Expressions

## ▶️ How to Run
python3 sentinelshield_waf.py

## 📊 Output
- Detects attack type
- Displays alert message
